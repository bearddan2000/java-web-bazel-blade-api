# java-web-bazel-blade-api

## Description
A POC for blade framework.

## Tech stack
- cors
- middleware
- endpoints
    - get, put, delete

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
